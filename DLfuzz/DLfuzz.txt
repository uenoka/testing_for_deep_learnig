# DLFuzz: Differential Fuzzing Testing of Deep Learning Systems

## URL
* Paper : https://arxiv.org/pdf/1808.09413.pdf
* Github : https://github.com/turned2670/DLFuzz

## どんなものか？
* 初の Differential fuzzing テストフレームワーク。
* DL system におかしな挙動をさせる。
* 入力を少しづつ変更してニューロンカバレッジを最大化する
* 元の入力と変更された入力で変更された予測の差を最大化する
* 手作業での食べリングやテストオラクルの相互参照を必要としない

## 先行研究と比べてどこがすごい？
* 異なる DL system のクロスリファレンスが不要
* 338% の敵対的入力を、89.82% 小さい摂動で生成した
* 平均して 2.86% 高いニューロンカバレッジを含んでいた
* 時間が 20.11% 短縮された
** これらは DeepXplore との比較
